# 🚀 SmartRH - Plateforme Web de Gestion des RH

**SmartRH** est une solution moderne et intelligente de gestion des ressources humaines, conçue spécifiquement pour les PME. Développée dans le cadre du projet de fin d'études **PI 2026** à l'**ENSUP**, cette application allie design "Premium" et architecture technologique de pointe.

## ✨ Fonctionnalités Clés
- **Authentification Sécurisée** : Gestion des sessions via NextAuth.
- **Tableau de Bord Dynamique** : Visualisation des KPIs (Effectifs, Paie, Congés).
- **Gestion des Employés** : Fiches détaillées, cycle de vie et contrats.
- **Module Paie & Grades** : Automatisation des bulletins de paie basés sur une structure salariale rigoureuse.
- **Circuit des Congés** : Workflow de demande et validation entre employé et manager.
- **Catalogue de Formations** : Suivi des compétences et participations aux sessions.

## 🛠️ Stack Technique
- **Framework** : Next.js 13+ (React)
- **Base de Données** : Prisma ORM avec PostgreSQL/MySQL
- **Styling** : Tailwind CSS 4 (Custom Teal Design)
- **UI Components** : Shadcn UI & Radix UI
- **Déploiement** : Cloud-ready (Render / Vercel)

## 📦 Installation & Démo
1. **Clonage** : `git clone https://github.com/mami572/SmartRH.git`
2. **Installation** : `npm install`
3. **Migration BD** : `npx prisma db push`
4. **Lancement** : `npm run dev` (Local) ou `npm run build && npm run start` (Production)
